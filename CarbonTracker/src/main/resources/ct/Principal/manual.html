Manual de Usuario - Carbon Tracker

Bienvenido al manual de usuario de Carbon Tracker. Aquí encontrarás una guía corta sobre cómo utilizar las funciones principales de la aplicación.

Pantallas Principales

La aplicación se divide en secciones accesibles desde el panel lateral izquierdo:

  • EMPRESAS: Permite ver listados, gestionar acciones y agregar registros.
  • EMISIONES: Gestión completa del cálculo de huella de carbono.
  • OPTIMIZACIÓN: Análisis inteligente y reportes.
  • ADMINISTRACIÓN: Gestión de usuarios y auditoría (Solo visible para administradores).

Gestión de Empresas

  • Ver Empresas: Muestra un listado de todas las empresas registradas. Desde aquí puedes buscar, editar, eliminar o ver las emisiones de una empresa específica.
  • Añadir Empresa: Abre un formulario para registrar una nueva empresa especificando nombre y sector.
  • Buscar Empresa: Usa la barra de búsqueda para filtrar la lista por nombre o sector.
  • Acciones de Empresa: En la columna "ACCIONES":
    • (Icono de Ojo): Ver todas las emisiones de esa empresa.
    • (Icono de Gráfico): Abrir Dashboard específico de esa empresa.
    • (Icono de Ubicación +): Gestionar Sedes/Sucursales.
    • (Icono de Lápiz): Editar nombre o sector.
    • (Icono de Canasta): Eliminar la empresa y sus emisiones.

Gestión de Emisiones

  • Ver Todas las Emisiones: Muestra un listado completo de los registros de emisiones.
  • Añadir Emisión: Abre un formulario para registrar una nueva emisión. Si eres administrador, podrás cambiar la empresa asignada; si eres empleado, se asignará a tu empresa o a la seleccionada.
  • Buscar Emisiones: Filtra por nombre de empresa, tipo o fecha.
  • Acciones de Emisión:
    • (Icono de Lápiz): Modificar valores de la emisión.
    • (Icono de Canasta): Eliminar permanentemente el registro.

Optimización

  • Centro de optimización: Listado seleccionable de empresas para análisis.
  • Análisis: Ofrece un informe personalizado de las áreas críticas y un plan de acción.
  • Imprimir / Descargar: Permite exportar el plan de acción en papel o formato TXT.

Gestión de Usuarios SOLO ADMIN

Panel exclusivo para controlar el acceso a la aplicación y los roles del equipo.

  • Lista de Usuarios: Muestra todos los usuarios registrados, su rol y su estado actual (ACTIVO en verde o BLOQUEADO en rojo).
  • Crear Usuario: Botón superior para dar de alta un nuevo usuario, asignando nombre, login, contraseña y rol.
  • Acciones sobre Usuarios:
    • (Icono de Lápiz): Editar nombre, cambiar rol o restablecer contraseña.
    • (Icono de Candado): Bloquear/Desbloquear. Un usuario bloqueado no podrá iniciar sesión (recibirá una alerta), pero sus datos se conservan.
    • (Icono de Canasta): Eliminar. Borra al usuario permanentemente.
      Nota de seguridad: Si el usuario tiene historial en la auditoría, el sistema impedirá el borrado para mantener la integridad de los datos. En ese caso, se recomienda bloquearlo.

Auditoría / Logs SOLO ADMIN

  • Registro de Seguridad: Tabla que recoge todas las acciones críticas realizadas en el sistema.
  • Campos:
    • FECHA/HORA: Momento exacto de la acción.
    • USUARIO: Quién realizó la acción.
    • ACCIÓN: Descripción detallada (ej: "Alta Sede", "Modificación Emisión").

Exportar a CSV

En las vistas de Empresas y Emisiones, el botón "Exportar Vista (CSV)" permite descargar los datos visibles (respetando los filtros de búsqueda aplicados) a un archivo compatible con Excel.
